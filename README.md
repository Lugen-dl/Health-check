
# Health-check скрипт

Скрипт для автоматической проверки контейнера

# Технический стек

- Bash, cron
- Docker compose
- Github actions



## Использование и установка

В этом скрипте был установленный временной промежуток при котором он запускается автоматически в actions. Если хотите запускать скрипт вручную, делаете это через GUI.

GUI
```bash
Actions -> .github/workflows/ci.yml -> Run worflows
```
